# Military Calisthenics — 4-Week Progressive Training Plan

A free, mobile-friendly workout tracker hosted on GitHub Pages. No app store, no subscription — just open the link and train.

🔗 **[Open the Workout Tracker](https://jeffreyklepper.github.io/workout)**

---

## What It Does

A fully interactive 4-week military calisthenics program you can run entirely at home with no equipment. Designed around the same principles used in military fitness training: bodyweight fundamentals, progressive overload, and a structured weekly split.

### Weekly Structure
| Day | Focus |
|---|---|
| Monday | Push (chest, shoulders, triceps) |
| Tuesday | Legs & Core |
| Wednesday | Active Rest |
| Thursday | Pull & Back |
| Friday | Full Body / Conditioning |
| Saturday | Core |
| Sunday | Full Rest |

### The 4 Weeks
- **Week 1 — Foundation:** Learn the movements, build form. 90 sec rest between sets.
- **Week 2 — Build:** Higher reps, new movements added. 75 sec rest.
- **Week 3 — Intensity:** Harder variations, timed circuits. 60 sec rest.
- **Week 4 — Peak:** Max effort, personal best tests, AMRAP retest.

## Features

- **Multi-profile support** — Select your name at launch; each person's progress is saved separately on their device
- **Per-exercise checkboxes** — Check off each exercise as you complete it
- **Auto day completion** — When all exercises in a day are checked, the day marks itself done automatically
- **Mark day done button** — Tap once to check all exercises and complete the day in one go
- **Real progress tracking** — Overall percentage and days completed update live as you work through the plan
- **YouTube how-to links** — Every exercise has a link to a tutorial video showing proper form, beginner to advanced
- **Reset individual days** — Undo a day's progress if needed
- **Full program reset** — Appears only at 100% completion so you can start the 4 weeks over fresh
- **Mobile friendly** — Works on any phone browser; bookmark it for quick access at home or in a hotel room

---

## How to Use

1. Open the link above on any device
2. Select your profile (or add your name the first time)
3. Pick the current week and day
4. Work through the exercises in order, checking each one off as you finish it
5. The day completes automatically when all exercises are checked
6. Come back anytime — your progress is saved in your browser

> **Note:** Progress is saved in your browser's local storage. It persists across visits on the same device and browser. Clearing your browser data will reset progress.

## No Equipment Needed

All exercises are pure bodyweight. A sturdy chair is helpful for tricep dips, step-ups, and Bulgarian split squats. The row variations use the edge of a table — no towels, bars, or additional props required.
